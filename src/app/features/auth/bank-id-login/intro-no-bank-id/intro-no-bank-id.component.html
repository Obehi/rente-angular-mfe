Renteradar har ingen automatisk løsning for innhenting av lånedata for brukere av BankID med passord. Hvis du vil bruke BankID med passord må du selv fylle inn dine lånedata. Renteradar trenger følgende informasjon:

  • Lånebeløp
  • Nominell rente
  • Løpetid
  • Termingebyr
  • Type lån

Identifiser deg nedenfor for å komme igang.
